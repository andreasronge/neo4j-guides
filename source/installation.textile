h3. Install Java


On Linux:
<code>
  apt-get install sun-java6-jdk sun-java6-jre
</code>

Also check out "this StackOverflow guide.":http://stackoverflow.com/questions/3747789/how-to-install-sun-jdk-on-ubuntu-10-10-maverick-meerkat

On Mac:

Mac comes with the JDK. For more information check out "this page.":http://developer.apple.com/library/mac/#documentation/Java/Conceptual/Java14Development/02-JavaDevTools/JavaDevTools.html

You should use java version 6 or greater (check @java -version@)
